## Overview

This is my personal portfolio website, inspired by the visual style of *Persona 3 Reload*. It showcases my projects, experience, and technical skills with a strong focus on backend development and automation.

The goal of this portfolio is to present real-world projects and demonstrate my transition from technical support into backend and automation engineering.

## Features

- Interactive UI inspired by *Persona 3 Reload*
- Project showcase with demos and detailed descriptions
- Sections for Experience, Skills, Education, and About Me
- Responsive design for desktop and mobile devices
- Clean and structured navigation

## Sections

- **Main Menu** – Entry point with navigation
- **Projects** – Highlighted backend and automation projects
- **Experience** – Professional background in systems and support
- **Skills** – Technologies organized by area (Backend, Systems, Frontend)
- **Education** – Certifications and continuous learning
- **About Me** – Summary of my background and career direction

## Tech Stack

- React
- JavaScript (ES6+)
- Tailwind CSS

## Purpose

This portfolio was built to showcase practical, real-world projects focused on:

- Backend development
- REST APIs
- Automation tools
- System-level problem solving
